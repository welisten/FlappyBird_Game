Flappy Bird

Este projeto tem como proposta a reprodução do famoso jogo Flappy Bird, utilizando HTML, CSS e JavaScript como ferramentas principais de desenvolvimento

Ao recriar o Flappy Bird, exploramos não apenas a diversão do jogo, mas também a oportunidade de aprimorar habilidades em programação web, promovendo uma experiência de aprendizado envolvente e prática.
